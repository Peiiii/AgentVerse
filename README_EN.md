## ğŸš€ Quick Start

### Requirements
- Node.js >= 18
- pnpm >= 8.0

### Installation

1. **Clone and Install**
```bash
git clone https://github.com/Peiiii/AgentVerse.git
cd AgentVerse
pnpm install
```

2. **Configure Environment**
```bash
cp .env.example .env
# Edit .env file to configure AI provider and API Key
```

3. **Start Development Server**
```bash
pnpm dev
```

Visit `http://localhost:3000` to use the application.
