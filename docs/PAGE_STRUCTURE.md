# 页面结构设计

```

├───────────────┬─────────────────────────┬───────────────────┤
│   会话列表     │      会话区域            │    会话成员        │

│ ┌───────────┐ │   ┌─────────────────┐   │ ┌───────────────┐ │

│ ┌───────────┐ │   └─────────────────┘   │                   │

│ │- 标题      │ │   │    消息列表      │   │ │[成员卡片]      │ │
│ │- 主题      │ │   │  [消息气泡]      │   │ │- 头像         │ │
│ │- 时间      │ │   │  - 头像         │   │ │- 名称         │ │
│ │- 成员数    │ │   │  - 名称         │   │ │- 角色         │ │

│               │   ┌─────────────────┐   │                   │

┌─────────────────────────────────┐
│     Agent管理弹窗                │

┌─────────────────────────────────┐
│     添加会话成员弹窗              │
└─────────────────────────────────┘

┌─────────────────────────────────┐

└─────────────────────────────────┘
```

## 关键设计说明

1. 三栏布局
   - 左侧：会话管理 (320px)
   - 中间：对话区域 (自适应)
   - 右侧：成员管理 (320px)

2. 信息展示
   - 核心信息直接展示
   - 次要信息通过 hover 展示
   - 配置通过弹窗管理
